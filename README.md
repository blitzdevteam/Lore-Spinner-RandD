# LoreSpinner

AI-powered interactive fiction platform. Transforms screenplay scripts into playable, branching narrative experiences with dynamic narration and player-driven choices.

**Stack:** Laravel 12 · Vue 3 · Inertia.js · Tailwind CSS 4 · PostgreSQL · OpenAI GPT-5.2

See [DOCUMENTATION.md](DOCUMENTATION.md) for full setup, deployment, and development guides.
